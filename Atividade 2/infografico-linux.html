Navegação

Mover-se pelo sistema de arquivos
ls
Listar arquivos e pastas do diretório
$ ls -la
Lista tudo com detalhes e arquivos ocultos
pwd
Mostrar o diretório atual completo
$ pwd
Retorna ex: /home/aluno/documentos
cd
Navegar para outro diretório
$ cd documentos
Entra na pasta "documentos"
cd ..
Voltar para a pasta anterior (pai)
$ cd ..
Sobe um nível na hierarquia de pastas
📁

Arquivos e Pastas

Criar, copiar, mover e remover
mkdir
Criar uma nova pasta
$ mkdir "minha pasta"
Aspas permitem espaço no nome
touch
Criar arquivo vazio rapidamente
$ touch arquivo.txt
Cria "arquivo.txt" vazio no diretório atual
nano
Editar arquivo de texto no terminal
$ nano arquivo.txt
Abre editor — Ctrl+O salva, Ctrl+X sai
cp
Copiar arquivo ou pasta
$ cp -r pasta/ backup/
Copia toda a pasta para "backup/"
mv
Mover ou renomear arquivo
$ mv velho.txt novo.txt
Renomeia ou move o arquivo
rm
Remover arquivo (sem lixeira!)
$ rm arquivo.txt
Apaga permanentemente — sem recuperação
rmdir
Remover pasta vazia
$ rmdir pasta/
Só funciona se a pasta estiver vazia
rm -rf
Remover pasta e todo conteúdo à força
$ rm -rf pasta/
⚠ Irreversível — use com muito cuidado!
👁

Visualização de Arquivos

Ler e inspecionar conteúdo
cat
Exibir conteúdo do arquivo no terminal
$ cat README.md
Mostra o conteúdo completo de uma vez
less
Visualizar com rolagem paginada
$ less log.txt
Navega com setas — q para sair
head
Ver as primeiras linhas do arquivo
$ head -n 10 arquivo.txt
Mostra as 10 primeiras linhas
tail
Ver as últimas linhas / monitorar em tempo real
$ tail -f log.txt
Acompanha o arquivo conforme cresce
🔍

Busca

Encontrar texto e arquivos no sistema
grep
Buscar texto dentro de arquivos
$ grep -i "erro" log.txt
Busca "erro" ignorando maiúsculas/minúsculas
find
Localizar arquivos no sistema
$ find . -name "*.txt"
Encontra todos arquivos .txt aqui
➤

Redirecionadores

Controlar o fluxo de saída de comandos
>
Redirecionar saída para arquivo (sobrescreve)
$ ls > lista.txt
Salva a saída do ls em lista.txt
>>
Redirecionar saída (adiciona ao final)
$ echo "ok" >> log.txt
Acrescenta sem apagar o conteúdo anterior
|
Pipe: passar saída de um comando para outro
$ ls | grep .txt
Filtra somente arquivos .txt na listagem
🔐

Permissões

Controlar acesso a arquivos
chmod
Alterar permissões de acesso do arquivo
$ chmod +x script.sh
Torna o script executável
chown
Alterar o proprietário do arquivo
$ chown aluno arquivo.txt
Muda o dono para o usuário "aluno"
⚙

Processos

Gerenciar o que está rodando
ps
Listar processos da sessão atual
$ ps
Mostra processos do usuário logado
ps aux
Listar todos os processos do sistema
$ ps aux
Exibe todos com PID, CPU e memória
top
Monitor de processos em tempo real
$ top
Mostra CPU e memória ao vivo (q = sair)
htop
Monitor interativo com interface visual
$ htop
Versão colorida e interativa do top
kill
Encerrar um processo pelo PID
$ kill -9 1234
Força encerramento do processo 1234
💻

Sistema e Shell

Informações e utilitários essenciais
whoami
Exibir nome do usuário atual
$ whoami
Retorna o usuário logado no momento
hostname
Ver o nome da máquina na rede
$ hostname
Exibe o nome do host configurado
uname
Ver informações do sistema operacional
$ uname -a
Mostra kernel, arquitetura e versão
lscpu
Ver informações da CPU
$ lscpu
Exibe arquitetura, núcleos e modelo
lsusb
Listar dispositivos USB conectados
$ lsusb
Mostra todos os periféricos USB
free -h
Ver uso de memória RAM
$ free -h
Mostra RAM total, usada e livre (-m = MB)
df -h
Ver espaço disponível em disco
$ df -h
Mostra uso do disco em GB/MB legível
ncdu
Analisar espaço em disco visualmente
$ ncdu
Interface interativa — mostra o que ocupa mais
echo
Imprimir texto ou variável no terminal
$ echo $HOME
Exibe o valor da variável de ambiente
man
Abrir o manual de qualquer comando
$ man ls
Mostra todas as opções do comando ls
sudo
Executar comando como administrador
$ sudo apt update
Atualiza pacotes com permissão root
history
Ver histórico de comandos digitados
$ history 10
Mostra os 10 últimos comandos usados
alias
Criar atalho para um comando longo
$ alias ll="ls -la"
Digitar "ll" executa "ls -la"
clear
Limpar a tela do terminal
$ clear
Equivalente ao atalho Ctrl + L
🌐

Rede e Internet

Conectar, baixar e acessar remotamente
hostname -i
Ver o endereço IP da máquina
$ hostname -i
Retorna o IP local do host
ip a
Ver interfaces de rede e endereços IP
$ ip a
Lista todas as interfaces com IPs e status
ping
Testar conexão com outro host
$ ping -c 4 google.com
Envia 4 pacotes e mede a latência
wget
Baixar arquivo da internet
$ wget https://site.com/arq.zip
Faz o download direto pelo terminal
curl
Transferir dados via URL (APIs, downloads)
$ curl https://api.exemplo.com
Faz requisição GET e exibe a resposta
ssh
Conectar a servidor remoto com segurança
$ ssh [email protected]
Abre sessão remota criptografada
📦

Compactação

Compactar e extrair arquivos
tar
Compactar ou extrair arquivos .tar.gz
$ tar -czf backup.tar.gz pasta/
Compacta a pasta em formato .tar.gz
zip
Compactar em formato .zip (multi-sistema)
$ zip -r projeto.zip pasta/
Gera .zip compatível com Windows
unzip
Extrair arquivo .zip
$ unzip arquivo.zip
Descompacta no diretório atual
